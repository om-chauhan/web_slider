# Web Slider Flutter

Flutter Web Slider Inspired by Microsoft Bing Search Engine 👇

## 🚀 Installation or Run server

Install, run the development server:

```bash
1. git clone https://github.com/om-chauhan/web_slider_flutter.git
2. flutter pub get
3. flutter run
```
